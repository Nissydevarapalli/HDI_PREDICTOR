# HDI Predictor

## Project Overview
The HDI Predictor is a Machine Learning project that predicts the Human Development Index (HDI) using socioeconomic indicators. The project includes data preprocessing, exploratory data analysis, model training, and a web application for prediction.

## Features
- Data preprocessing
- Exploratory Data Analysis (EDA)
- Machine Learning model training
- HDI prediction using a trained model
- User-friendly web interface

## Technologies Used
- Python
- Flask
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Project Structure

```
HDI_PREDICTOR/
├── dataset/
├── model/
├── src/
├── requirements.txt
└── README.md
```

## Installation

1. Clone the repository
2. Install the required packages

```bash
pip install -r requirements.txt
```

3. Run the application

```bash
python src/app.py
```
